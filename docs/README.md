# 装配式钢结构网页版报价工具

这个目录用于 GitHub Pages 发布。

文件：
- `index.html`
- `styles.css`
- `app.js`

当前报价逻辑：
- 材料采购成本 A = 主体钢结构 + 围护系统 + 功能件 + 机电系统
- 材料附加费 B = A × 材料附加费率
- 服务成本 C = 服务费用
- 管理费 D = (A + B + C) × 管理费率
- 利润 E = (A + B + C + D) × 利润率
- 报价 = A + B + C + D + E
- 折扣 = 按折扣率或手动折扣额计算
- 成交价 = 报价 - 折扣
- 年度维保费 = 成交价 × 维保费率
- 总维保费 = 年度维保费 × 质保年限

如果要让页面上线，请在 GitHub 仓库设置中打开 Pages，并把发布源设为：
- Branch: `main`
- Folder: `/docs`
